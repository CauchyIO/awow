Branches are generated from the issue identifier, not invented manually.

| Context | Pattern | Example |
|---|---|---|
| Linear | `{user}/{TEAM}-{number}-{slug}` | `alex/proj-315-add-tax-columns-to-export` |
| Azure DevOps | `issue/{TEAM}-{number}` | `issue/PROJ-1042` |
| Jira | `{user}/{TEAM}-{number}-{slug}` | `alex/proj-42-fix-loading` |
| GitHub Issues | `{user}/issue-{number}` | `alex/issue-123` |
| Feature / long-lived | `feature/{descriptive-slug}` | `feature/payments-gateway-rewrite` |

Most board tools generate this automatically.
