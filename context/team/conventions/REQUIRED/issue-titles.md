## Pattern (TODO — fill via /setup-awow)

Verb-first, specific enough that the title alone tells you what the issue delivers.

| Pattern | Example | When to use |
|---|---|---|
| `Implement {thing}` | Implement OAuth flow for service principal auth | New capability, net-new code |
| `Add {thing} to {surface}` | Add Board-First section to copilot-instructions.md | Extending an existing artefact |
| `Fix {symptom} in {area}` | Fix dashboard loading bug in revenue KPI notebook | Bug with a known symptom |
| `Update {thing}` | Update streaming job tags to include `job_type` | Modifying existing behaviour |
| `Remove {thing}` | Remove deprecated DLT pipeline resources | Decommission or cleanup |
| `Draft {thing}` | Draft §8 cross-team visibility section | Content or documentation |
| `Define {thing}` | Define board tagging convention reference | Convention, policy, or standard |

Vague titles ("OAuth", "Fix bug") fail this convention.
