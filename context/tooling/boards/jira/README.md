# Jira — reference index

> **v0.1 status:** skeleton. Reference is the same shape as Linear's; depth lands in v0.2 once we have engagement data from a Jira-resident team.

Jira is the most-used board in enterprise. The mapping is more variable than Linear or ADO because Jira projects are heavily customised per team.

## When `/setup-awow` infers Jira

Board URL hostname matches `*.atlassian.net`.
