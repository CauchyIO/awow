# .agents/commands/seed/

Seed-phase commands. The three a team adopts in the first cycle. `/setup-awow` wires these up by default.

| Command | What it does |
|---|---|
| `refinement-prep.md` | Drafts a feature for the next refinement from a brief, slidedeck, or transcript |
| `process-workitem.md` | Router for board work items — classify, validate, plan, apply, validate, document |
| `process-transcript.md` | Gated pipeline for meeting transcripts — parse, classify, extract, propose, execute |

A team that runs these three for one cycle has shipped the Seed phase. The Spread commands become reasonable next steps.
