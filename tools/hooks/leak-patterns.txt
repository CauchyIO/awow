# Leak signatures scanned by tools/hooks/pre-push before content reaches a remote.
#
# This repo is PUBLIC. Reports derived from agent session traces (mlflow-export,
# awow-usage-coach, prompt-skill-analysis, project-retrospective) carry customer/
# session data — real names, private issue IDs, infra topology, cost figures. They
# must never be committed here. These patterns are a backstop, not a guarantee.
#
# One Perl-compatible (PCRE) regex per line. Lines starting with # are comments.
# Keep patterns HIGH-SIGNAL: a false positive that blocks a legit push gets the
# whole hook disabled. Test a new pattern with:
#   git grep -nP 'YOUR_PATTERN'
# and make sure it does NOT match existing tracked content.
#
# Escape hatch for a verified-clean push:  git push --no-verify
# or:  AWOW_SKIP_LEAK_SCAN=1 git push

# Absolute local path to a repo other than awow (session exports embed these).
/Users/[^/\s]+/repos/(?!awow[/\s"'`])

# A concrete per-experiment MLflow export subdir committed into content
# (the export dir followed by a slash and a name). Bare "mlflow_export" prose
# without a trailing path segment does not match.
mlflow_export/[A-Za-z0-9._-]+

# Money figures with cents and a currency — spend/cost data from cost reviews.
\b[0-9][0-9,]*\.[0-9]{2} ?(?:EUR|USD)\b

# Destructive Terraform command transcripts (incident-forensics reports).
terraform\s+apply\b[^\n]*--?auto-approve
