# .agents/commands/spread/

Spread-phase commands. Adopted after the team has run Seed for one cycle and a second engineer wants to pick up the workflow.

| Command | What it does |
|---|---|
| `board-skill.md` | Recipes for auto-linking commits to issues, status-nudging, board-side automations |
| `sprint-review-prep.md` | Drafts a sprint review from the cycle's board activity and commits |
| `release-notes.md` | Generates release notes from issues moved to Done in a given range |

Adopt via `/awow-add <name>`. The wizard runs the prerequisite check and writes a one-page brief before installation.
