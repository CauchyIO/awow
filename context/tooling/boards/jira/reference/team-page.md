# Jira — team page conventions reference (skeleton)

Jira projects have a description; many enterprise teams also keep a Confluence space.

## What a team profile should contain

- **Mission** — one sentence; mirror of `context/team/mission.md`.
- **Members** — name, role, focus area.
- **Sprint cadence** — length, start day.
- **Links** — to the team's Confluence space, dashboards, runbooks.

## Wizard responsibilities

**Mode A.** Draft a project description + Confluence space landing page from Steps 2–4 answers. Emit as manual paste (Atlassian MCP write surface for project description is uneven).

**Mode B.** Read the existing project description and any linked Confluence space.
